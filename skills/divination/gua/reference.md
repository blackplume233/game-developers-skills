## 乾坤特殊辞

- **乾·用九**：见群龙无首，吉。
- **坤·用六**：利永贞。

## 爻位关系速查

| 术语 | 含义 |
|------|------|
| **当位/得正** | 阳爻居奇位（初、三、五）或阴爻居偶位（二、四、上） |
| **中** | 居二（下卦之中）或五（上卦之中） |
| **应** | 初↔四、二↔五、三↔上，阴阳相应为有应 |
| **承** | 阴爻在阳爻之下，承托之 |
| **乘** | 阴爻在阳爻之上，乘凌之 |
| **比** | 相邻两爻的关系 |
| **九五** | 阳刚中正，最尊之位 |
| **六二** | 阴柔中正，最顺之位 |
